# Keyboard Shortcuts

The complete keyboard shortcut reference is available in **Settings → Keyboard Shortcuts**. That tab shows all registered shortcuts grouped by context and is always up to date with the current version of the app.

## Global Shortcuts

These shortcuts work anywhere in the application, regardless of focus:

| Shortcut | Action |
|---|---|
| ⌘K / Ctrl+K | Open omnibar |
| ? | Show keyboard shortcut cheatsheet |
| [ | Toggle sidebar navigation |
| ⌘P / Ctrl+P | Pause active session |
| ⌘D / Ctrl+D | Delete active session |

## Session Shortcuts

These shortcuts apply when a session is focused or selected:

| Shortcut | Action |
|---|---|
| ⌘↵ / Ctrl+Enter | Accept / approve pending approval request |
| ⌘R / Ctrl+R | Resume paused session |

## Omnibar Shortcuts

These shortcuts are active when the omnibar is open:

| Shortcut | Action |
|---|---|
| ↑ / ↓ | Navigate suggestions |
| Enter | Confirm selected action |
| Tab | Switch to creation form |
| Escape | Close omnibar |

## Re-Triggering the Shortcut Overlay

Press **?** (without any modifier) when no text input is focused to open the full shortcut overlay. Press **Escape** to close it.

## Customizing Shortcuts

Keyboard shortcut customization is not yet supported in the UI. You can view the full registry in **Settings → Keyboard Shortcuts** to see all registered shortcuts and their contexts.
